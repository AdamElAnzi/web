Hi ha 4 maneres de unir CSS amb HTML

  • Carpeta CSS amb un arxiu que es digui styles.css

    • Aquesta és la manera que més utilitzarem.

  • Posar en el head la etiqueta style i a dins posar body{} i entre {} posar background-color: +el color que es vulgui posar
  • Ficar un h2 per exemple el següent: style:"background-color: red"> Utilitzar l'etiqueta style en una etiqueta HTML
  • Utilitzar la regla @import, s'ha de posar: @import url(css/styles.css); entre styles al head.No s'ha de fer a no ser que tinguem diferents codis css i els volguem importar tots en un de sol. És una mala praxis i per tant no ho farem
