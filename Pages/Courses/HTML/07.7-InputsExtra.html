Input Extra

Input tipus color

Un input de tipus color permet triar un color en una paleta de colors, si feu una pagina on l'usuari hagi d'escollir un color per a algun item en concret o el color de fons d'una pagina us anira molt be

Input de tipus number

Ens deixa posar numeros. Les petites fletxes que surten a la dreta per pujar i baixar el numero es diuen widgets, Aquest input s'ha d'utilitzar per temes numerics que no siguin dates, per demanar dates ja tenim d'altres

Input de tpus range

Ens pinta una barra que podem modificar per seleccionar un valor. Accepta l'atribut step que determina de quant en quant es mou la nostra barra. També té l'atribut min i max

Input de tipus reset

Es crea un boto que permet borrar tots els camps del form
