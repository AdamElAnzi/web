INPUTS PER LES DATES

Els inputs per posar dates són diversos i s'han de controlar. Alguns no funcionen en alguns navegadors i s'han de conèixer.

EXEMPLES D'INPUTS PER LES DATES

Un input de tipus date

És un camp que ens deixa introduir un dia, un mes i any. Funciona bé en tots els navegadors. A sota el codi, teniu el resultat en el fitxer html.

        
            <form>
                <input type="date">
            </form>

Un input de tipus datetime

Està obsolet. Es comporta com un camp de tipus text. No s'ha d'utilitzar.

        
            <form>
                <input type="datetime">
            </form>

Un input de tipus datetime-local

És un camp que permet entrar un dia, un mes, any i l'hora. No funciona en Mozilla Firefox.

        
            <form>
                <input type="datetime-local">
            </form>

Un input de tipus time

És un camp que permet entrar la hora. Funciona en tots els navegadors.

        
            <form>
                <input type="time">
            </form>

Un input de tipus time

Si voleu demanar dia i hora en un formulari i que funcioni en tots els navegadors, haurieu de posar un input de tipus date i un input de tipus time

        
            <form>
                <input type="date">
                <input type="time">
            </form>

Un input de tipus month

Si voleu demanar un mes en concret. També heu d'introduir l'any.

        
            <form>
                <input type="month">
            </form>
